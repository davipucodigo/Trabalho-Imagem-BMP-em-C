<h1>Manipulando BMP</h1>
Neste trabalho criamos 4 funções para manipular imagens:
 - Separar cores;
 - Corte X e Y;
 - Colocar uma região da imagem, delimitada por X e Y em cinza;
 - Gerar um ASCII da Imagem.

 As 3 primeiras foram desemvolvidas pelo Henrique e o Gerador de ASCII da imagem por Davi.
